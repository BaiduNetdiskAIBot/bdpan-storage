# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.4.0] - 2026-04-07

### Added

- 语义搜索：不记得文件名？用自然语言描述就能找到文件，告别精确关键词烦恼
- 文件复制：支持在网盘内快速复制文件或文件夹
- 文件移动：轻松整理目录结构，将文件移动到指定位置
- 文件重命名：直接在 Agent 中对网盘文件进行重命名操作

## [1.3.0] - 2026-04-01

### Added

- 大文件分片下载功能

### Fixed

- 修复下载超时问题

## [1.2.0] - 2026-03-23

### Added

- 增加搜索、移动、复制、重命名、创建文件夹功能。

## [1.1.2] - 2026-03-23

### Fixed

- 稳定性：修复 v1.1.1 引入的若干问题，提升文件操作命令的整体可靠性。

## [1.1.1] - 2026-03-21

### Fixed

- 稳定性：修复 v1.1.0 中新增的搜索、移动、复制、重命名、创建文件夹功能中的边界问题。

## [1.1.0] - 2026-03-20

### Added

- 搜索：支持通过关键词搜索百度网盘指定目录中的文件和目录。
- 移动：支持将文件或目录移动到百度网盘指定目录中的其他位置。
- 复制：支持在百度网盘指定目录内复制文件或目录。
- 重命名：支持对百度网盘指定目录的文件和目录进行重命名操作。
- 创建文件夹：支持在百度网盘指定目录中创建新目录。
