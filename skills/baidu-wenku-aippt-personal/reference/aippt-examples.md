# AI PPT 生成示例

本文档包含 `bdpan aippt` 命令的使用示例。

## 命令格式

```bash
bdpan aippt "<用户原始query>" "<title>" "<page>" "<style>" "<scene>" "<color>"
```

**参数说明：**

  - **第1参数（用户query）：** 严格使用用户的原始输入，不做任何加工或改写
  - **title（标题）：** Agent 从用户 query 中提取或生成一个简短精炼的 PPT 标题
- **page（页数）：** 1-10、11-20、21-30、31-40、40+（默认：11-20）。根据用户指定的页数选择最匹配的枚举值
- **style（风格）：** 创意趣味、年终总结、卡通手绘、扁平简约、文艺清新、中国风、企业商务、未来科技、文化艺术
- **scene（场景）：** 晚会表彰、传统节日、婚姻爱情、生日祝福、医学科普、建筑报告、工作总结、年终总结、工作汇报
- **color（颜色）：** 绿色、蓝色、紫色、橙色、黄色、红色、黑色、白色

> Agent 根据用户 query 自动生成 title，并判断最佳 page/style/scene/color 取值；无法判断时使用默认值（page=11-20，style=扁平简约，scene=工作汇报，color=蓝色）。
> **title/page/style/scene/color 参数对用户不可见**，Agent 静默选择后直接执行，不向用户展示这些参数。
